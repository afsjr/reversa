# Design System

**Comando:** `/reversa-design-system`
**Fase:** Cualquiera
**Analogía:** El estilista

---

## Qué hace

El estilista cataloga el guardarropa del sistema: paleta de colores, tipografía, espaciados, tokens de diseño. Las "reglas de moda" que gobiernan la apariencia del proyecto.

Útil cuando necesitas reescribir la interfaz o crear nuevos componentes manteniendo consistencia visual con lo que ya existe.
